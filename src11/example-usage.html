Ejemplo de Uso - PROJECT CONTROL Dashboard

1. Estructura de Datos

El dashboard espera recibir un objeto con la siguiente estructura:

const projectData = { duration: [ { duration: 9 }, // Duración en días de cada tarea { duration: 5 }, { duration: 7 }, { duration: 5 }, { duration: 6 } ], employees: [ { name: 'Empleado', salary: null }, // Nombre y salario del empleado { name: 'Albert', salary: 50 }, { name: 'Rose', salary: 45 } ], materials: [ { name: 'Concrete' }, // Lista de materiales { name: 'Steel' }, { name: 'Wood' } ], extraCosts: [ { name: 'Transport', value: 400 }, // Costos adicionales { name: 'Equipment', value: 300 }, { name: 'Consultation', value: 250 } ], realCost: 12500, // Costo real del proyecto totalCost: 14700 // Costo total presupuestado };

2. Cómo Usar la Función

Para cargar datos en el dashboard, usa la función initializeProjectDashboard():

// Cargar datos personalizados initializeProjectDashboard(projectData);
